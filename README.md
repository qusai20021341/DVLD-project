# DVLD - Driver's Vehicle Licensing Department System

A comprehensive **Desktop Application** for managing a Driver and Vehicle Licensing Department. Built with **C# .NET Windows Forms** using 3-Tier Architecture.

![DVLD Banner](project%20Images/gov.jor.dvld-header.png)

## 📋 Project Overview

DVLD is a full-featured management system designed to handle all operations of a Driver's License Department, including managing people, applications, licenses, tests, and users.

### ✨ Features

- **User Authentication** (Login System with different roles)
- **People Management** (Add, Update, Delete, Find persons)
- **Driver Management**
- **License Applications** (New, Renew, Replacement, etc.)
- **Local & International Driving Licenses**
- **Driving Tests Management** (Vision, Written, Practical)
- **License Detain & Release**
- **User Management** (Admin panel)
- **Application Types & Test Types Configuration**

## 🏗️ Architecture

This project follows **3-Tier Architecture**:

- **Data Access Layer (DAL)** - Database operations
- **Business Layer (BL)** - Business logic and validation
- **Presentation Layer** - Windows Forms UI

## 🛠️ Technologies Used

- **C#** + **.NET Framework**
- **Windows Forms** (WinForms)
- **SQL Server**
- **ADO.NET**
- **3-Tier Architecture**
- **OOP Principles**
